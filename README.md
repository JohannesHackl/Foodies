# Recipes-WebPage

A lightweight, browser-based recipe application built with HTML, CSS, and JavaScript.  
The site allows users to browse recipes, add new ones, manage a pantry, and maintain a shopping list — all through a simple and responsive interface.

## Features
- **Recipe browsing** — View available recipes with images and descriptions.
- **Add new recipes** — Interactive form powered by JavaScript (`insert_recipe_script.js`).
- **Pantry management** — Track ingredients you have at home.
- **Shopping list** — Maintain and update items you need to buy.
- **Animated navigation bar** — Smooth JS‑based navbar interactions.

## Tech Stack
- **HTML** — Page structure  
- **CSS** — Styling and layout  
- **JavaScript** — Interactivity and dynamic content  

## Project Structure
- `index.html` — Landing page  
- `recipes.html` — Recipe overview  
- `recipeinsert.html` — Add‑recipe interface  
- `pantry.html` — Pantry view  
- `shoppinglist.html` — Shopping list  
- `scripts/` — JavaScript logic (e.g., navbar animation, recipe insertion)  
- `imgs/` — Image assets  
- `style.css` — Global styling
